# Harness engineering cho HelpID

Thư mục này chứa tài liệu markdown tiếng Việt để agent làm việc hiệu quả với repo HelpID Android.

## Cách dùng nhanh

1. Đọc `../AGENTS.md` trước.
2. Đọc `tong-quan-kien-truc.md` để hiểu repo.
3. Chọn tài liệu theo phần đang sửa:
   - Android: `android.md`
   - Web/API: `web-api.md`
   - Bảo mật/dữ liệu y tế: `bao-mat-du-lieu.md`
   - Quy trình coding: `quy-trinh-code.md`
   - Quy trình test: `quy-trinh-test.md`
4. Trước khi trả lời người dùng, chạy qua `checklist-truoc-khi-tra-loi.md`.

## Mục tiêu

- Giảm thời gian agent đọc lại repo từ đầu.
- Chuẩn hóa cách build, test và review thay đổi.
- Nhắc các rủi ro đặc thù của app khẩn cấp: SMS, vị trí, dữ liệu y tế, public profile link, offline sync.
- Giữ tài liệu và kết quả làm việc bằng tiếng Việt theo yêu cầu của repo.

## Nguyên tắc cập nhật

- Khi thêm module, dependency, command hoặc luồng bảo mật mới, cập nhật tài liệu liên quan trong thư mục này.
- Không biến tài liệu thành nhật ký dài. Chỉ ghi điều có ích cho lần coding/testing sau.
- Nếu tài liệu và code lệch nhau, code là nguồn sự thật; sau đó cập nhật tài liệu.
